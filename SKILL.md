# 日本語論文ポリッシュ (jp-paper-polish)

ユーザーが日本語の論文・レポートを貼り付けたとき、以下のルールで書き直す。

---

## 最重要ルール：文体の方向性

**目標：普通の大学生が自分で書いたような、自然で読みやすい文章**

- 長い一文は2〜3文に分ける
- 難しい漢語・複合語は平易な言葉に替える
- 同じ意味なら短い方を選ぶ
- 論文らしい硬さは残しつつ、AI特有の「流暢すぎる滑らかさ」は崩す

---

## 禁止する記号・表現（AI特有パターン）

以下は使わない：

| 禁止 | 代替 |
|------|------|
| 「 」（鍵括弧による強調） | 使わないか、そのまま表記 |
| ——（ダッシュ） | 。で文を切る、またはそのまま続ける |
| · （中黒の箇条書き） | 通常の箇条書きか、文章で表現 |
| ※ の多用 | 注記は（）か脚注で |
| まず〜次に〜最後に（過剰な列挙） | 自然な流れで繋ぐ |
| 〜という点において | 〜について |
| 〜であることは言うまでもない | 削除するか「〜である」 |
| 〜を踏まえると | 〜を考えると |
| 〜に他ならない | 〜である |
| 〜と言えよう | 〜と思われる / 〜だろう |
| 〜の観点から | 〜から見ると |
| 多岐にわたる | 様々な / いくつかの |
| 顕著に | はっきりと / 大きく |
| 不可欠である | 必要である / 大切である |
| 寄与する | 役立つ / 影響を与える |
| 看過できない | 無視できない |

---

## 長文分割のルール

- 1文が60字を超えたら分割を検討する
- 「〜し、〜し、〜することで」のような連続する節は、一度文を切る
- 関係節（〜という〜）が重なるときは、先に短く説明文を置く

---

## よくある文法ミスの修正

- は・が・を・に の助詞の誤用
- 〜れる / 〜られる の受身・可能の混用
- 〜てしまう の不自然な多用
- 句読点の位置（読点の付けすぎ・なさすぎ）
- 敬体（です・ます）と常体（だ・である）の混在

## 修正例（Before / After）

### 1｜禁止語彙：多岐にわたる → 様々な

**Before:**
> 本研究では、AIが社会に与える影響について多岐にわたる側面から考察した。

**After:**
> 本研究では、AIが社会に与える様々な影響について考察した。

---

### 2｜禁止語彙：顕著に → はっきりと / 大きく

**Before:**
> 実験結果において、処理速度の向上が顕著に確認された。

**After:**
> 実験結果では、処理速度がはっきりと向上した。

---

### 3｜禁止語彙：不可欠である → 必要である

**Before:**
> 持続可能な社会の実現には、技術革新と政策の両立が不可欠である。

**After:**
> 持続可能な社会を実現するためには、技術革新と政策の両立が必要である。

---

### 4｜禁止語彙：寄与する → 役立つ / 貢献する

**Before:**
> このアルゴリズムは計算効率の向上に寄与することが期待される。

**After:**
> このアルゴリズムは計算効率の向上に役立つと考えられる。

---

### 5｜禁止語彙：〜に他ならない → 〜である

**Before:**
> この結果は、従来の理論が限界を迎えていることの証左に他ならない。

**After:**
> この結果は、従来の理論が限界を迎えていることを示している。

---

### 6｜禁止語彙：〜を踏まえると → 〜を考えると

**Before:**
> 以上の考察を踏まえると、今後の研究では定量的な分析が求められる。

**After:**
> 以上の考察を考えると、今後の研究では定量的な分析が求められる。

---

### 7｜長文分割：複数の節をひとつの文にまとめない

**Before:**
> 近年、環境問題が深刻化する中で、持続可能な開発目標（SDGs）の達成に向けた取り組みが国際社会において重要視されており、各国が協力して解決策を模索している。

**After:**
> 近年、環境問題が深刻化している。そのため、持続可能な開発目標（SDGs）の達成が国際社会で重視されるようになった。現在、各国が協力して解決策を探している。

---

### 8｜長文分割：〜し、〜し、〜することで の連鎖を切る

**Before:**
> データを収集し、分析し、可視化することで、ユーザーの行動パターンを把握することができた。

**After:**
> データを収集・分析して可視化した。その結果、ユーザーの行動パターンを把握できた。

---

### 9｜接続詞の過剰列挙：まず〜次に〜最後に → 自然な流れ

**Before:**
> まず、先行研究を調査した。次に、仮説を立てた。最後に、実験を通じて検証した。

**After:**
> 先行研究を調査した後、仮説を立て、実験を通じて検証した。

---

### 10｜AI的な接続詞：〜であることは言うまでもない → 削除 or 言い換え

**Before:**
> 言語モデルの発展が近年の自然言語処理分野を大きく変えたことは言うまでもない。

**After:**
> 言語モデルの発展は、近年の自然言語処理分野を大きく変えた。

---

### 11｜鍵括弧による強調を外す

**Before:**
> 本研究では「持続可能性」を中心的な概念として用いる。また、「技術的負債」という問題も論じる。

**After:**
> 本研究では持続可能性を中心的な概念として用いる。また、技術的負債という問題についても論じる。

---

### 12｜〜と言えよう → 〜と思われる / 〜だろう

**Before:**
> この傾向は、今後さらに強まると言えよう。

**After:**
> この傾向は、今後さらに強まると思われる。

---

### 13｜〜の観点から → 〜から見ると / 〜について

**Before:**
> 教育の観点から見ると、この政策には問題点が多い。

**After:**
> 教育という面から見ると、この政策には問題点が多い。

---

### 14｜序論の過剰フォーマル構造 → 自然な導入

**Before:**
> 本論文の目的は、〜を明らかにすることである。本論文の構成は以下の通りである。第1章では〜を述べ、第2章では〜を分析し、第3章では〜について考察する。

**After:**
> 本論文では、〜について明らかにすることを目的とする。以下、〜の分析と考察を順に述べる。

---

### 15｜結論の過剰まとめ表現 → 簡潔に

**Before:**
> 以上の考察を総合すると、本研究において得られた知見は多岐にわたり、今後の研究に対して多大な寄与をなすものと思料される。

**After:**
> 本研究で得られた知見は、今後の研究に役立つと考えられる。

---

## 表達参考庫（学術表現リスト）

### 接続表現（文頭・節頭）

**順接・結論**

| 表現 | ニュアンス |
|------|-----------|
| そのため | 原因→結果（最頻出） |
| したがって | 論理的帰結 |
| このため | やや口語的な因果 |
| これにより | 手段→結果 |
| その結果 | 実験・調査の後に |
| 以上のことから | まとめの導入 |
| これらのことから | 複数根拠のまとめ |

**逆接・対比**

| 表現 | ニュアンス |
|------|-----------|
| しかし | 一般的な逆接 |
| 一方 / 一方で | 対比の提示 |
| これに対して | 先行内容への反論・比較 |
| 他方 | やや硬め、論文向き |

**付加・補足**

| 表現 | ニュアンス |
|------|-----------|
| また | 情報の追加（最汎用） |
| さらに | 程度・内容の強調追加 |
| 加えて | 並列追加、やや硬め |
| なお | 補足・注意事項の導入 |
| 併せて | 同時進行・付随事項 |

**言い換え・まとめ**

| 表現 | ニュアンス |
|------|-----------|
| すなわち | 定義・言い換え |
| つまり | 要約・噛み砕き |
| 以上のように | 節末のまとめ |
| 今後の課題として | 展望・限界の導入 |

### 文末パターン

**考察・判断**

| パターン | 使いどころ |
|---------|-----------|
| 〜と考えられる | 根拠ある推論（最頻出） |
| 〜と思われる | やや弱い推測 |
| 〜と推察される | データから間接的に推論 |
| 〜と判断した | 研究者が積極的に結論を出す |

**実験・観察結果**

| パターン | 使いどころ |
|---------|-----------|
| 〜が示された | データが示す（受動） |
| 〜が確認された | 実験での確認 |
| 〜が明らかになった | 新知見の提示 |
| 〜が示唆される | 間接的・弱い示唆 |
| 〜が認められた | 統計的・客観的な確認 |

**目的・研究設計**

| パターン | 使いどころ |
|---------|-----------|
| 〜を目的とする | 序論での研究目的 |
| 〜を行った | 手法・実験の記述 |
| 〜を検討した | 考察の動詞 |
| 〜を提案する | 手法提案 |
| 〜を明らかにした | 結論での成果 |

---

### 動詞フレーズ（研究行為の記述）

```
分析した / 評価した / 比較した / 検証した / 考察した
調査した / 測定した / 観察した / 実施した / 試みた
開発した / 構築した / 設計した / 実装した / 提案した
に着目した / に基づいて〜 / に焦点を当てた
```


---

## 出力フォーマット

1. **修正後の本文をそのまま出力する**（説明は最小限）
2. 大きく変えた箇所があれば、末尾に簡潔なメモをつける（任意）
3. ユーザーが「どこを直したか教えて」と言った場合のみ、差分を詳しく説明する

---

## 作業の流れ

1. テキストを受け取る
2. 文ごとに上記ルールをチェックする
3. 書き直した全文を出力する
4. 「他に直してほしい箇所はありますか？」と一言添える
